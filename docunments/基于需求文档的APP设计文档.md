## 🧩 功能模块设计

### 1. 动态手语识别与翻译模块

**功能流程**：

1. 用户点击“开始识别”按钮。
2. 启动摄像头，实时捕获手语视频。
3. 视频上传调用本地模型进行连续动态识别。
4. 将识别结果翻译为自然语言文本。
5. 在界面中央区域实时显示翻译结果。

**设计要求(目前做不到捏)**：

- 识别延迟 ≤ 500ms。
- 支持整句输出，符合手语语法。
- 识别过程中提供视觉反馈（如边框高亮、识别中动画）。

---

### 2. 多模态交互与对话模块

**功能流程**：

- **手语 → 文字/语音**：
  - 识别手语后，自动显示文字。
  - 用户可设置是否启用语音播报。
- **语音 → 文字**：
  - 健听人士点击“语音输入”按钮。
  - 调用语音识别 SDK，将语音转为文字。
  - 文字显示在对话区域中。
- **对话历史**：
  - 自动记录本次会话中的所有对话（手语翻译 + 语音识别）。
  - 支持滚动查看历史记录。

**界面组件**：

- 主界面分为：
  - 上半部分：摄像头预览
  - 下半部分：手语翻译结果 + 对话历史记录（双方交替显示）

### 设置页面内容扩展

| 设置项         | 选项/说明 |
|----------------|-----------|
| 语言设置       | 选择输出语言（中文、英文等） |
| 模型设置       | 选择识别模型（标准版/轻量版） |
| 播报设置       | 开启/关闭语音播报、音量、语速 |
| 振动反馈       | 开启/关闭识别成功振动 |
| 文本大小       | 小/中/大/超大 |
| 语音识别引擎   | 讯飞/百度/系统默认 |
